# 🎲 3D Dice Roller App

A stylish, interactive **3D dice rolling application** built with Flutter.  
This app not only simulates a realistic dice roll with a satisfying 3D animation but also includes a fun guessing game.  
It features a sleek, modern UI with **dynamic themes**, **sound effects**, and **smooth performance** across all devices.

---

## ✨ Features

- 🎞️ **Realistic 3D Animation:** Beautiful, smooth “tumble” animation for every dice roll.  
- 🎯 **Guessing Game:** Guess the outcome before rolling and get instant feedback (right/wrong popup).  
- 🔊 **Sound Effects:** Includes a dice roll sound for a more immersive experience.  
- 🌗 **Dynamic Theming:** Automatic light/dark mode support with gradient backgrounds.  
- 🧩 **Custom App Icon:** Clean and modern launcher icon.  
- 📱 **Responsive Design:** Looks great on phones and tablets of all sizes.
